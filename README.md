# 🍃 Leaf Chat 🍃

## 🌿 About 
* A chat app that you can chat to any user, and global chat. 

## ⭐ Note:
* Only available on Mobile Display


## 🌿 Screenshots


## ⭐ Available Features
* Login/SignUp Authentication
* Reset password
* Logout
* Display all user registerd
* User/Global chat rooms
* Upload avatar
